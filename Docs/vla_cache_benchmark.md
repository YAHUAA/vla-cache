## 结论

1. **推理加速**: VLA-Cache 稳态单步推理延迟从 ~109ms 降至 ~83ms，**加速比约 1.32x**
2. **计算量**: GPU TFLOPs 从 1.91 降至 1.49，**计算量降低约 22%**，说明 Cache 有效跳过了部分 transformer 层的冗余计算
3. **首步开销**: 启用 Cache 后首步延迟略微增加（222ms vs 210ms，+5.7%），这是因为需要初始化 cache 结构
4. **成功率**: 启用 VLA-Cache 后 10/10 成功（无 Cache 时 8/10），单次 trial 存在随机性，但说明 Cache 机制未引入任务性能退化
